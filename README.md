# Predicting Human-to-Human Transmissibility of Viruses

This project uses machine learning models to predict whether viruses can infect humans and spread between them.

## Methods

- Feature engineering
- SMOTE and sampling techniques
- Stratified cross-validation
- Hyperparameter tuning

## Models

- Logistic Regression
- Random Forest
- XGBoost

## Evaluation Metrics

- PR-AUC ≈ 0.90
- ROC-AUC ≈ 0.92
- F1 Score
- Balanced Accuracy

## Tools

- Python
- Scikit-learn
- Pandas
- NumPy
